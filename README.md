## 🚀 Getting Started
If you'd like to explore the code locally, follow these steps:

### Prerequisites
Make sure you have Node.js and npm (Node Package Manager) installed on your machine.

### Installation
1. Clone the repository:
    git clone https://github.com/itzyeboiabraham/portfolio-v2-react.git

2. Navigate into the project directory:
    cd portfolio-v2-react

3. Install the required dependencies:
    npm install

4. Start the development server:
    npm start

*(Note: If you used Vite to bootstrap the React app, run `npm run dev` instead).*

## 📂 Repository Structure
* `.vscode/` - Editor configuration settings.
* `public/` - Static assets that are served directly.
* `src/` - Contains all the React components, assets, and styling files.
* `.gitignore` - Specifies untracked files to be ignored by Git (e.g., node_modules).
* `index.html` - The main entry point of the application.
* `package.json` & `package-lock.json` - Lists project dependencies, versions, and scripts.
* `vite.config.js` - Configuration file for the Vite build tool.
